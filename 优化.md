### 4. 限流（单机限流和多机限流）

限制用户调用的总次数
限制用户调用的频率 （限制用户每两秒调用一次 以及如果在一分钟内调用超过10次封禁用户）

分布式限流:

1. 把用户的调用信保存在redis中
2. 在网关中对用户进行限流例如：spring cloud gateway

限流算法：

1. 令牌桶算法
2. 漏桶算法
3. 滑动窗口算法
4. 固定窗口算法
